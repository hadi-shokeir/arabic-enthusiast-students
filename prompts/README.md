# AI Prompt Library

These prompts are versioned outside the API code so the teaching behavior can be reviewed, improved, and rolled back without digging through serverless logic.

Runtime API entry points:

- `exercise-feedback.md` grades open-ended Arabic answers and explains mistakes.
- `conversation-partner.md` powers level-aware Arabic conversation practice.
- `lesson-recommender.md` suggests daily practice from progress, mistakes, and SRS due items.
- `pronunciation-feedback.md` compares expected text with browser speech transcription.
- `lesson-plan.md` helps Hadi generate a practical multi-week private lesson plan.

Every prompt must account for:

- Arabic track: MSA, Levantine, or both.
- Student level and current goals.
- Feedback language preference.
- Cost: use short, structured outputs unless a tutor-facing task needs depth.

Arabic generated by these prompts should be reviewed by Hadi before it becomes fixed curriculum content.
